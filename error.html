                    • Front-End Web Developer – Angular, React.js, vue.js

                     

                    DETAILS

                    Location: Addison , TX

                    Position Type: Direct-Hire / Contract to Hire

                    Salary Conversion: BOE!

                    Citizenship Requirements:  US Citizens and those authorized to work in the US are encouraged to apply. No sponsorship is being offered at this time.

                     

                    ABOUT THE POSITION

                    GTN Technical Staffing is seeking a Front-End Web Developer for a Direct-Hire or Contract to Hire opportunity in Addison, TX. The Front-End Web Developer will join a team developing and maintaining web applications leveraging C# / ASP.NET framework with a heavy JS front-end. The Front-End Web Developer will build and implement modern, rich UIs utilizing JavaScript frameworks such as Angular, react.js, or vue.js. Candidates need to have developed and deployed into production one application built with AngularJS, React.js, or vue.js.

                    The Front-End Web Developer should possess solid understanding of delivering SaaS-based solutions with a good understanding of AWS services (S3, CloudFront, etc) as well as solid experience with RESTful APIs.

                     

                    TECHNICAL REQUIREMENTS

                    5+ years of experience in the following skill sets…

                    • UI Development
                      • At least 1 production implementation with AngularJS -or- React.js -or- vue.js
                      • Strong JavaScript coding ability
                      • HTML / HTML5 / CSS / CSS3
                    • Front-End Build Processes
                      • Grunt / gulp.js
                      • webpack

                    PREFERRED

                    • C# / ASP.NET Web Development / Design / Architecture
                      • NET Core Experience
                    • Entity Framework
                    • Cloud / SaaS-based Development
                      • AWS Services
                      • S3 Storage
                      • Content Delivery Network (CDN) – Amazon CloudFront
